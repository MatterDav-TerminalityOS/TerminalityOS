# TerminalityOS

This is a custom terminal-style OS  which is built in C# and please know that it has no bootloader or kernel.

I was inspired by Terry A. Davis to make an OS and note I'm 12 years old

-- Features --
- A fake Bios Boot.
- A ASCII Boot Animation.
- clear, echo, exit, real ls and cd, ascii for directory, help, a follow command for my tikok account and a version command.
- Almost FULL ASCII Directory and echo echoes in random colors
- Cool Animations
- And Finally, almost 500 lines of code

 - Built with:
 - C#
 - .NET
 - Visual Studio
 - And a bit of AI
